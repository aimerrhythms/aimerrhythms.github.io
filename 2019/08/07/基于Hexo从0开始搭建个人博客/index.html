基于Hexo从0开始搭建个人博客(基于windows操作系统)

安装node.js

网址：nodejs.org 下载LTS版本，安装

node.js安装包提供了node.js和npm包管理器

安装Git（将博客部署到Github上时需要安装）

网址：git-scm.com/downloads 下载Windows版本，安装

开始在本地搭建个人博客

打开GitBash终端，查看版本信息：

由于国内镜像源很慢，所以先利用npm安装cnpm:

npm install -g cnpm --registry=http://registry.npm.taobao.org

安装完成后

cnpm -v

开始正式安装Hexo：

cnpm install -g hexo-cli

安装完成后

hexo -v

使用Hexo搭建博客

在本地磁盘创建Blog文件夹（之后的步骤出现问题直接删除Blog文件夹重来即可）

使用Hexo生成博客

hexo init

启动博客

hexo s

在浏览器中打开本地地址：http://localhost:4000

这样本地博客就搭建完成了，接下来把博客部署到Github上公开使用，以后可以通过Github地址访问个人博客
